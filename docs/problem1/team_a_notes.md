# 队员 A 论文写作提示

可以围绕下面 4 部分组织论文：

## 1. 问题抽象

- 把 64 支队伍看成待分配对象
- 把 16 个小组看成容量均为 4 的容器
- 将题目要求拆分为硬约束和软约束

## 2. 算法设计

- 先随机把 11 个市级队放入 11 个不同小组
- 再按“约束更紧的县级队优先”进行分配
- 对每支县级队，从可行组中优先选同市队更少、人数更少的小组
- 多次随机搜索，保留多个可行方案

## 3. 评价体系

- 软约束违反次数
- 来源城市分散度
- 小组实力均衡性

## 4. 结果展示

- 展示最优方案分组表
- 展示前若干方案评分图
- 比较不同权重下方案变化
