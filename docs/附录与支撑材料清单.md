## 3 不建议放入附录的内容

根据竞赛要求，**赛题本身已提供的数据不要放入附录**。因此：

- `试题发布版/2026problem.pdf`
- 学校封面原件

这些文件不建议再放入论文附录正文中。

## 4 最后提交前检查

1. 所有脚本是否能运行。
2. 论文中的每个关键数字是否能在结果文件中找到来源。
3. 支撑材料中的数据是否为自主查阅和整理所得。
4. 是否已删除可能暴露身份的信息。
